關於 Canvas

Canvas 是 HTML5 的新元素，可以形容為一個畫布容器，本身無繪製能力，可透過 JavaScript 繪製圖形。
包含繪圖、文字、合成圖照片、建立動畫、甚至處理即時的影片播放、遊戲及動畫製作。
